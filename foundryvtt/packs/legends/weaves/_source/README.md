# Weaves Compendium

Spells (weaves) for the Legends TTRPG magic system.

## Status

To be populated with weave definitions.

## Structure

Each weave should include:
- Name
- Weave Type (simple/complex)
- Action Cost (1 or 2 actions)
- Primary Energy (type and cost)
- Supporting Energy (if any)
- Effect Type (damage, healing, buff, control, etc.)
- Range
- Target
- Duration
- Save Type
- Description
