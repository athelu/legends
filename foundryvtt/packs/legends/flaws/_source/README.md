# Flaws Compendium

Character flaws for the Legends TTRPG system.

Flaws are disadvantages or limitations that characters can take to gain additional trait points.

## Status

To be populated with flaw definitions.

## Structure

Each flaw should include:
- Name
- Point Value (how many trait points it grants)
- Flaw Type (physical, mental, social, etc.)
- Description of the disadvantage
