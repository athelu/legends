# D8 TTRPG - Part 9: Backgrounds

## Background System

Each character chooses one background during character creation. Backgrounds represent your life before adventuring and provide:
- Skill Training: A set of skills with specific ranks (48 XP value)
- Starting Equipment: Basic tools and gear for your profession
- Background Feature: A special ability or social benefit (primarily
  roleplay-focused)

### Background Patterns

All backgrounds provide exactly 48 XP worth of skills, distributed in one of four patterns, plus additional XP for customization:
- Pattern A (Generalist): 4 skills at rank 2, +24 XP to spend
- Pattern B (Focused): 1 skill at rank 3 + 2 skills at rank 1, +36 XP to spend
- Pattern C (Balanced): 2 skills at rank 2 + 2 skills at rank 1, +40 XP to spend
- Pattern D (Specialist): 1 skill at rank 3 + 1 skill at rank 2, +32 XP to spend
